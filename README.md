# RTOS-Based Vehicle Safety System 🚗🛡️

Smart car safety system using FreeRTOS on ARM Cortex-M4 (TM4C123GH6PM).

## 🌟 Features
- **Real-Time Park-Assist:** Ultrasonic sensors with varying alarm frequencies.
- **Speed-Triggered Door Locking:** Automatic lock when exceeding a safe threshold.
- **Collision Avoidance:** Active braking and alert mechanisms.

## ⚙️ RTOS Task Architecture
Built with FreeRTOS using preemptive scheduling, prioritizing critical safety tasks (collision detection) over auxiliary tasks (dashboard updates).

## 🛠️ Technologies
`C` `FreeRTOS` `ARM Cortex-M4` `TM4C123GH6PM` `Keil/CCS`
